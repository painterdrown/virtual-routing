# virtual-routing

 > 基于 Go 语言的虚拟路由程序。

## Usage

  + 获取项目到本地

    `go get -u github.com/painterdrown/virtual-routing`

  + 编译安装可执行文件

    `cd $GOPATH/src/github.com/painterdrown/virtual-routing && go install`

  + 运行程序

    `virtual-routing <ip> <port>`

    将 \<ip\> 替换成你想要设置的 ip 地址，可以是任意的 ip 地址，只是起到标识作用，与实际的 ip 无关；
    将 \<port\> 替换成你想要监听的端口号。
